## Navicat Cloud 是什么？先搞清楚它不是什么

很多人第一次听到 Navicat Cloud，第一反应是"这是不是一个在浏览器里用的在线数据库工具？"

不是。

Navicat Cloud 本质上是 Navicat 系列桌面软件的一个**云端协作组件**。你还是照样用桌面端，但你的连接设置、SQL 查询、数据模型、虚拟组这些东西，会自动同步到云上。

换句话说，它做的是"把你的工作环境帮你保存好、随时能取出来"这件事。

这对以下几类人来说非常实用：

- 经常在多台电脑之间切换的开发者
- 有远程协作需求的小团队
- 不想每次换电脑都重新配一堆连接的 DBA
- 手头有 iPad 或 iPhone 想偶尔查一下数据的工程师（Navicat Cloud 也支持 iOS）

## 核心功能一次说清楚

**跨设备自动同步**

连接设置、查询、模型、虚拟组，在 Windows / macOS / Linux / iOS 之间全部自动同步。登录账号，一切就绪，不需要手动导出导入。

**256-bit AES 加密 + SSL 传输**

所有文件存储在 Amazon S3 上，AES 加密落盘，通过 SSL 通道传输。安全这块，是认真做的。

**两步验证（2FA）**

账号多一层保护，防止意外泄露。如果你存了很多生产环境的连接信息，这一点挺重要。

**团队项目协作**

可以创建项目、添加成员、分配角色（管理员、编辑、只读等），大家在同一个项目里同步工作，活动日志实时记录谁改了什么。

**Navicat Cloud Portal**

一个 Web 界面，用来集中管理文件、用户、项目和云服务状态。管理员视角，看得清楚。
